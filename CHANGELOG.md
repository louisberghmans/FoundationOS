# Changelog

All notable changes to FoundationOS are documented here.

## 1.0.0 — 2026-07-16

- Replaced the browser-local prototype with a real Node 24 and SQLite application.
- Added genuine first-run setup, invitation-only membership, roles, sessions, CSRF, throttling, and audit events.
- Added theory-of-change objectives and append-only evidence observations.
- Added the accessible opportunity action board with server-enforced stage transitions.
- Added immutable non-objection decision rounds: Support, Neutral, or Object, with no implied consent.
- Added funds, fiscal budgets, multi-currency and multi-year commitments, agreements, amendments, installments, payments, refunds, cancellations, and reversals.
- Added private, versioned documents with checksums and authorized download routes.
- Added meeting availability, agenda records, grant reviews, and learning records.
- Added a one-container Docker deployment with persistent `/data`, health checks, a read-only root filesystem, and multi-architecture image publication.
- Replaced the MIT license with GPL-3.0-or-later and documented third-party provenance.
- Removed all family-derived seed data and made clean setup the default.
